# 19-Budget-Tracker

## Purpose
This is a Progressive Web Application(PWA) project that will track a persons budget. This site gives the user a fast and easy way to track their money whether they're connected to the internet or not.

## Built-With
- JavaScript
- MongoDB 
- Node
- Express 
- Mongoose
- IndexedBD
- Service Worker
